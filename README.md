# AI Credit Memo Generator

A Next.js app that generates credit risk memoranda using GPT-4.

## Setup

1. Install dependencies:
   ```
   npm install
   ```

2. Add a `.env.local` file:
   ```
   OPENAI_API_KEY=your_openai_api_key_here
   ```

3. Run the development server:
   ```
   npm run dev
   ```

## Deploy

You can deploy this app on Netlify or Vercel. Be sure to add your environment variable in the project settings.

Enjoy!
